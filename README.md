# 💰 SaaS Financeiro com WhatsApp

Sistema de gestão financeira pessoal integrado ao WhatsApp com IA.
O usuário envia mensagens como "Gastei R$50 no mercado" e o sistema
registra, categoriza e analisa automaticamente.

---

## Tecnologias

| Camada | Tecnologia | Versão |
|---|---|---|
| Backend | Python + FastAPI | 3.14+ |
| Banco de dados | PostgreSQL | 16+ |
| Cache e filas | Redis | 7+ |
| Frontend | Flutter | 3+ |
| IA | LangChain + OpenAI | - |
| Containers | Docker + Compose | 29+ |
| Mensageria | WhatsApp Cloud API | - |
